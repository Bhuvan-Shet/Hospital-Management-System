# Hospital Management System

The Hospital Management System is a web-based application developed to manage hospital operations efficiently. The system helps hospitals handle patient records, doctor management, appointments, and administrative tasks through a simple and user-friendly interface.

This project is designed to reduce manual work and improve the management of hospital data digitally.

## Features

* Patient Registration and Management
* Doctor Management
* Appointment Booking System
* Admin Dashboard
* Secure Login Authentication
* Database Management
* Responsive User Interface

## Technologies Used

* Python
* Flask / Django
* HTML
* CSS
* JavaScript
* SQLite / MySQL
* Git & GitHub

## Objective

The main objective of this project is to simplify hospital management processes by providing an efficient digital system for handling patient and hospital information.
